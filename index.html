HTML

  • HTML refers to a base coding language used to give a webpage structure
  • HTML stands for Hypertext Markup Language and is composed of two parts: the head and the body
  • The head element contains information about the webpage
  • The body elements refers to the visible content of the webpage
  • When working in HTML, every tag is to be enclosed within angle brackets
  • Remember that end tags include a forward slash tat the start of them
  • Metadata is data relating to our webpage that contains information integral for it to function
  • HTML attributes define any additional information relating to an element within the webpage
  • Child elements refer to elements nested within other elements
  • Parents elements are elements that have child elements nested within them
  • Semantic HTML elements describe the role that an element has on a webpage
  • Heading elements are ranked in size from 1 to 6 with 1 being the largest and 6 being the smallest

CSS

  • CSS is a coding language used to make a webpage more visually appealing
  • CSS stands for Cascading Style Sheets
  • The most common uses of CSS are inline CSS, Internal CSS Style Sheets and External CSS Style Sheets
  • Inline CSS is CSS that is included within a specific tag
  • Internal Style Sheets are CSS instructions include within the header of an HTML document
  • External Style Sheets are seperate documents linked to the HTML document within its header, between the meta and title tags
  • CSS declations contain the property, which is the element being addressed, and the value, which is the desired change made
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an area
  • Class selectors, creeated by adding a period to the start of a designated word, allow CSS commands to be put on specific HTML elements

Git

  • Git commands are, like bash commands, commands issued on the command that always begin with git and are useful in tracking and maintaining progress with code
  • git status checks what branch we're on
  • When working in branches the original branch is always refered to as main
  • git checkout -b branch-name creates a new branch and automatically switches to it
  • git add adds modifications to our current working branch
  • Using -A after git add allows for all changes to be added
  • git commit is essentially a way of saving changes made within a piece of code
  • Adding -m to git commit allows you to attach a customised name/message
  • git pull allows you to pull an external branch's modifications into your current one
  • git push allows you to push changes made in your current branch into an external one

JavaScript

  • JavaScript is a coding language used to for creating website logic
  • Javascript files can be linked to HTML files within the footer
  • A variable is a named container that allows us to store data in our code
  • When adding variables, the equals sing is used to assign the specific data that we want
  • Arrays are variables used to hold groups of data
  • Strings are lists of the data we want stored within a variable
  • Methods are actions that can be performed on allocated data
  • Control flow is the order in which a computer executes code in a script
  • Chrome DevTools can be accessed by right clicking within your webpage and selecting inspect
  • if statements are conditional statements that allow our code to make decisions based on criteria
  • Adding else to the start of if statements allows you to have multiple criteria
  • Criteria being met is referred to as truthy
  • Indexes are unique numbers assigned to data within an array
  • For loops use predictable patterns in indices to allow a piece of code to be continuously executed
  • Functions are sets of instructions for your computer to follow
